
# MathPulse - Real-time Multiplayer Math Game

MathPulse is a high-speed real-time multiplayer math game built with Next.js and Firebase. Challenge your friends to solve equations faster than their heart can pulse!

## Features

- **Real-time Multiplayer**: Powered by Firebase Firestore for instantaneous synchronization.
- **Custom Question Pools**: Hosts can build their own challenges with auto-calculating answers.
- **Dynamic Difficulty**: Hardcore random mode with 3-digit multiplication and complex division.
- **Celebratory Leaderboard**: Top 3 podium finish to crown the math champion.
- **Mobile Responsive**: Designed for the fastest fingers on any device.

## Getting Started

1. **Clone the repository.**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Setup Firebase:**
   - Configure your `.env.local` with Firebase credentials.
   - Enable Firestore and Authentication.
4. **Run the app:**
   ```bash
   npm run dev
   ```
